# NBA Doha map handover

Files:
- index.html: live map
- hotspots.js: hotspot content
- manager.html: visual hotspot editor

Workflow:
1. Open manager.html on GitHub Pages.
2. Add or edit hotspots.
3. Download hotspots.js.
4. Upload it to GitHub and replace the existing hotspots.js.
5. Commit and refresh the live map.

## Hotspot icons

The manager includes a visual icon picker. Selecting a venue type chooses a suggested icon, which can be overridden. The exported `hotspots.js` stores an `icon` value for each location.
